# Templates de e-mail (Banners AFFEMG)

Cada arquivo corresponde a um template em **Authentication > Emails > Templates** no painel do Supabase.
Para cada um: cole o **Subject** no campo de assunto e o conteudo do arquivo `.html` no corpo (Source/HTML).

A logo vem de `assets/email-logo.png`, que precisa estar publicado (isso acontece no deploy).

| Template no painel | Arquivo | Subject |
|---|---|---|
| Invite user | `invite.html` | Seu acesso aos Banners AFFEMG foi aprovado |
| Reset Password | `reset-password.html` | Redefinir sua senha dos Banners AFFEMG |
| Confirm signup | `confirm-signup.html` | Confirme seu e-mail nos Banners AFFEMG |
| Magic Link | `magic-link.html` | Seu link de acesso aos Banners AFFEMG |
| Change Email Address | `change-email.html` | Confirme seu novo e-mail nos Banners AFFEMG |
| Reauthentication | `reauthentication.html` | Seu c?digo de confirma??o |
